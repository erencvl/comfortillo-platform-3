Comfortillo - Mental Health Support Platform (MVP Prototype)

Comfortillo is an early stage mental health support platform designed to offer anonymous emotional sharing, community interaction, and empathetic AI-assisted support within a safe and minimalistic interface.

This repository showcases the UI/UX, core layouts, navigation flow, and component architecture of the platform. It currectly functions as a frontend-only MVP created to demonstrate the product vision and user experience.

*Features (UI/UX Showcase Only):*

- Category based emotional sharing
- Post cards, reactions and basic interactions
- Reporting modal + moderation UI

*AI Assisted Support (Placeholder):*

-Chat UI designed for 24/7 empathic support
-Predefined conversation starters
-API integration is planned for future versions

*Auth System (Prototype):*

- Login / Register modal
- Profile page layout
- Editable user profile fields
- No backend / database - data does not persist yet

*Design & User Experience:*

- Fully custom UI
- Responsive layout
- Soft, calming color palette
- Component-based architecture (React + Next.js + Tailwind)
- Current Development Status:

This project is an MVP level prototype.

*The following features are NOT implemented yet:*

- Real authentication
- Database/persistent user data
- AI backend or API
- Email verification
- Secure login sessions
- Admin panel & moderation tools
- This systems will be added in future development stages.
